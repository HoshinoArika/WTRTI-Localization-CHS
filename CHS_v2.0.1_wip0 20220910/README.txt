--------------------------------
FM mode
--------------------------------
This mode contains additional indicators, that work only in "Test Flight":
     Excess power, W
     Drag, kgf (lbf)
     Total mass, kg(lb)
     No fuel mass, kg(lb)
     Thrust to Weight ratio
     Critical G, %

To make these indicators to work, you need to find the "debug" section in the "config.blk" file (in the game root folder) and to add a new line:
   enableFMCommands:b=yes

--------------------------------
VR mode
--------------------------------
VR support implemented with SteamVR, requires it to be installed.
SteamVR Controller Bindings support. Allows you to assign VR controller buttons to specific WTRTI actions. To make settings to be available in SteamVR, you need first go in a "Test Flight" or a Battle, and then select WTRTI from the list of programs in the SteamVR's Controller Bindings window.

For Oculus users:
To make the game to run through SteamVR, you have to set the compatibility mode to Windows 7, for aces.exe in win32 and win64 folders.

Hotkeys:
    CTRL-ALT-V - toggles "HMD tracking" mode.
    CTRL-ALT-Z - toggles "Left controller tracking" mode.
    CTRL-ALT-X - toggles "Right controller tracking" mode.
    CTRL-ALT-S - saves overlay's position and rotation from "tracking" modes to the no "tracking" mode.
    SHIFT-ALT-R - resets overlay's position and rotation(x:0.0, y:0.0, z:-0.5).
    SHIFT-ALT-<LEFT/RIGHT> - moves overlay on X axis.
    SHIFT-ALT-<UP/DOWN> - moves overlay on Y axis.
    SHIFT-ALT-<PAGE UP/PAGE DOWN> - moves overlay on Z axis.
    SHIFT-ALT-Q - decrease overlay's size.
    SHIFT-ALT-E - increase overlay's size.

--------------------------------
Режим FM
--------------------------------
Этот режим содержит дополнительные индикаторы, работающие только в "Пробном вылете":
     Избыточная мощность, Вт (Excess power, W)
     Лобовое сопротивление (Drag, kgf)
     Общая масса, кг (Total mass, kg)
     Масса без топлива, кг (No fuel mass, kg)
     Тяговооружённость (Thrust to Weight ratio)
     Критическая перегрузка, % (Critical G, %)

Также, чтобы данные индикаторы работали, необходимо в файле "config.blk"(в папке с игрой), найти раздел "debug" и добавить новую строку:
   enableFMCommands:b=yes

--------------------------------
Режим VR
--------------------------------
Поддержка VR реализована с помощью SteamVR.
"SteamVR Controller Bindings", позволяет назначать кнопки VR контроллера на определенные действия WTRTI. Чтобы настройки были доступны в SteamVR, необходимо сначала зайти в "Пробный вылет", а затем выбрать WTRTI из списка программ в окне Controller Bindings в SteamVR.

Для пользователей Oculus:
Чтобы игра запустилась через SteamVR, необходимо установить режим совместимости с Windows 7, для aces.exe в папках win32 и win64.

    CTRL-ALT-X - режим "Прикрепить к правому контроллеру".
    CTRL-ALT-S - сохранить позицию оверлея из режима "Прикрепить к", в свободном пространстве.
    SHIFT-<ALT-R - сброс позиции оверлея (x:0.0, y:0.0, z:-0.5).
    SHIFT-<ALT-<ВЛЕВО/ВПРАВО> - перемещение оверлея по оси X.
    SHIFT-<ALT-<ВВЕРХ/ВНИЗ> - перемещение оверлея по оси Y.
    SHIFT-<ALT-<PAGE UP/PAGE DOWN> - перемещение оверлея по оси Z.
    SHIFT-<ALT-Q - уменьшить размер оверлея.
    SHIFT-<ALT-E - увеличить размер оверлея.

--------------------------------
FM模式
--------------------------------
这个模式包含了只在"试飞"中工作的额外指示器:
     多余能量（Excess power）, W
     阻力, 千克力（磅力）
     总质量, 千克（磅）
     无油空重, 千克（磅）
     推重比（Thrust to Weight ratio）
     临界G值（Critical G）, %

要使用这些指示器, 你需要在"config.blk"文件中找到 "debug" 段落(在游戏根目录，也就是战争雷霆的游戏安装目录“WarThunder”)然后加入下面一行代码:
   enableFMCommands:b=yes

--------------------------------
VR模式
--------------------------------
需要先安装好SteamVR才能正常使用，因为本应用的VR的支持是通过它实现的.
SteamVR 控制器绑定支持. 允许你设定VR控制器按钮来执行WTRTI的特定操作. 
若要让这些设定在SteamVR中可用，你需要先进入“试飞”或者一场战斗，然后在从SteamVR的控制器绑定窗口的程序列表中选择WTRTI.

Oculus用户:
让游戏通过SteamVR来运行，你需要设置“aces.exe”文件的兼容性模式为Windows 7，包括在win32和win64两个文件夹中的文件.

Hotkeys:
    CTRL-ALT-V - 开关 "HMD 跟踪" 模式.
    CTRL-ALT-Z - 开关 "左控制器跟踪" 模式.
    CTRL-ALT-X - 开关 "右控制器跟踪" 模式.
    CTRL-ALT-S - 从“跟踪模式”中保存屏幕覆盖的位置和旋转到“不跟踪模式”.
    SHIFT-ALT-R - 重置屏幕覆盖的位置和旋转(x:0.0, y:0.0, z:-0.5).
    SHIFT-ALT-<左/右> - 延X轴移动屏幕覆盖.
    SHIFT-ALT-<上/下> - 延Y轴移动屏幕覆盖.
    SHIFT-ALT-<PAGE UP/PAGE DOWN> - 延Z轴移动屏幕覆盖.
    SHIFT-ALT-Q - 增大屏幕覆盖尺寸.
    SHIFT-ALT-E - 减小屏幕覆盖尺寸.
